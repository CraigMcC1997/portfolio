3D Collision Detection Using the GJK Model (2019-2020)

My honours project was the development of a 3D collision detection system which implemented the Gilbert-Johnson-Keerthi collision detection algorithm. It is written in C++17 and developed using Visual Studio 2019. This project showcases 3D collisions between different shapes, each shape comprising of a mesh, a convex hull for detections and a controller class for movements.
